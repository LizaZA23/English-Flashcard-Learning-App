## Проблемы
1) Отсутствие подсказок для новых пользователей.
Новые пользователи могут испытывать трудности с пониманием всех функций
- **Решение:** Добавить встроенное руководство или подсказки при первом запуске

2) Ограниченные возможности массового добавления карточек.
Нет функции импорта карточек из файлов
- **Решение:** Реализовать импорт из CSV/Excel форматов

3) Нет подтверждения при удалении карточки.
Карточка удаляется сразу без подтверждения
- **Решение:** Добавить диалог подтверждения удаления

## Предложения по улучшению
1) Расширенная статистика
- **Описание:** Добавить график прогресса по времени, heat map активности, чтобы был более глубокий анализ эффективности обучения

2) Категории наборов
- **Описание:** Возможность группировать наборы по темам или уровням, чтобы было улучшение организации больших коллекций

3) Мобильное приложение
- **Описание:** Разработать мобильную версию для изучения в любое время и для значительного расширения аудитории
