# План тестирования English Flashcard Master

## 1. Функциональное тестирование

### Цели:
- Проверить корректность работы CRUD операций
- Валидировать алгоритм интервальных повторений
- Проверить сохранение и загрузку данных из PostgreSQL
- Убедиться в правильности расчетов статистики

### Области тестирования:
- Управление наборами карточек
- Операции с карточками (добавление, редактирование, удаление)
- Режим изучения и алгоритм повторений
- Система прогресса и статистики
- Работа с базой данных

### Инструменты:
- Ручное тестирование
- PostgreSQL 14
- Java 11
- JavaFX 17

## 2. Пользовательское тестирование (Usability Testing)

### Цели:
- Оценить удобство интерфейса
- Выявить сложности в навигации
- Проверить интуитивность основных функций
- Собрать предложения по улучшению

### Методология:
- Тестирование с 5 одногруппниками
- Сессии по 15-20 минут
- Выполнение конкретных задач
- Анкетирование после тестирования

### Задачи для пользователей:
1. Создать новый набор карточек
2. Добавить 3 карточки с словами
3. Пройти сессию изучения
4. Просмотреть статистику прогресса
5. Найти определенную карточку

## 3. Критерии успеха
- Все CRUD операции работают корректно
- Алгоритм повторений рассчитывает даты правильно
- Интерфейс интуитивно понятен для новых пользователей
- Отсутствуют критичные ошибки при работе с данными
