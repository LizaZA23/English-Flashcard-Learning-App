Flashcard Learning App — это десктопное приложение на JavaFX, предназначенное для эффективного изучения английской лексики с использованием методики интервальных повторений (Spaced Repetition System), которая оптимизирует время повторения материала для долгосрочного запоминания. Приложение ориентировано на студентов, изучающих английский язык, преподавателей, создающих учебные материалы, а также самостоятельно обучающихся пользователей, стремящихся систематизировать свой словарный запас. Ключевые функции включают создание тематических наборов карточек, интеллектуальную систему повторений, визуализацию прогресса обучения, поиск и фильтрацию карточек, а также надежное хранение данных в PostgreSQL.

Предварительные требования
----------------------------------------
Java JDK 11 или выше
PostgreSQL 12+
Maven 3.6+

Пошаговая установка
Клонирование репозитория

bash
git clone https://github.com/your-username/flashcard-learning-app.git
cd flashcard-learning-app
Настройка базы данных

sql
-- Создание базы данных
CREATE DATABASE flashcard_app;

-- Подключение к базе
\c flashcard_app;

-- Создание таблиц (скрипты в database/schema.sql)

Отредактируйте файл src/model/DatabaseConnection.java:

Сборка проекта

bash
mvn clean compile
Запуск приложения

bash
mvn javafx:run
Альтернативный запуск через IDE
Импортируйте проект как Maven проект

Примеры использования
Создание набора карточек
Перейдите на вкладку "Мои наборы"
Введите название набора (например, "Базовые глаголы")
Укажите тему (например, "Грамматика")
Нажмите "Добавить набор"

Добавление карточек
java
// Пример программного добавления
FlashcardSet set = new FlashcardSet(0, "Базовые глаголы", "Грамматика", null);
set.save();

Flashcard card1 = new Flashcard(set.getId(), "run", "бегать");
Flashcard card2 = new Flashcard(set.getId(), "eat", "есть");
set.addFlashcard(card1);
set.addFlashcard(card2);

Режим изучения
Выберите набор для изучения из выпадающего списка
Используйте клавиши:
Пробел - перевернуть карточку
1/Y - помню слово
2/N - не помню слово
