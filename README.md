# English Flashcard Master

### Описание Проекта

**English Flashcard Master** — это десктопное приложение на JavaFX, предназначенное для эффективного изучения английской лексики с использованием методики интервальных повторений (Spaced Repetition System), которая оптимизирует время повторения материала для долгосрочного запоминания. Приложение ориентировано на студентов, изучающих английский язык, преподавателей, создающих учебные материалы, а также самостоятельно обучающихся пользователей, стремящихся систематизировать свой словарный запас. Ключевые функции включают создание тематических наборов карточек, интеллектуальную систему повторений, визуализацию прогресса обучения, поиск и фильтрацию карточек, а также надежное хранение данных в PostgreSQL.

## Установка и Запуск

Убедитесь, что у вас установлены:

- **Java Development Kit (JDK) v11+**
- **Apache Maven v3.6+**
- **PostgreSQL v12+**
- **JavaFX SDK v17+**

### Пошаговая Инструкция

#### 1) Клонирование репозитория
```bash
git clone [https://github.com/yourusername/english-flashcard-master.git](https://github.com/LizaZA23/English-Flashcard-Learning-App.git)
cd english-flashcard-master
```

#### 2) Создание и Конфигурация Базы Данных

1. Создайте базу данных в PostgreSQL:
```sql
CREATE DATABASE flashcard_app;
```

2. Обновите файл конфигурации `model/DatabaseConnection.java` соответствующими параметрами подключения:
```java
private static final String URL = "jdbc:postgresql://localhost:5432/flashcard_app";
private static final String USER = "your_postgres_user";
private static final String PASSWORD = "your_postgres_password";
```

#### Сборка и Запуск Проекта

1. Используйте Maven для сборки проекта:
```bash
mvn clean install
```

2. Запустите приложение через IDE, запустив главный класс `Main.java`

После успешного запуска откроется главное окно приложения с тремя вкладками для работы с карточками.

## Примеры Использования

- **Создание набора карточек**.
Перейдите на вкладку "Мои наборы" → Введите название и тему набора → Нажмите "Добавить набор"

- **Добавление карточек**.
Выберите набор → Введите английское слово в поле "Вопрос" → Введите перевод в поле "Ответ" → Нажмите "Добавить карточку"

- **Изучение слов**.
Перейдите на вкладку "Режим изучения" → Выберите набор для изучения → Используйте пробел для переворота карточки → Нажимайте "Помню"/"Не помню" для оценки запоминания

- **Просмотр прогресса**.
Перейдите на вкладку "Прогресс" → Выберите набор для просмотра статистики → Анализируйте график распределения по сложности
